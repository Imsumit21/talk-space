   ```bash
   cd client && npm install @fontsource/inter@^5.2.8 @fontsource/space-grotesk@^5.2.10 colord@^2.9.3 tailwindcss-animate@^1.0.7
   ```

2. Replace the contents of `client/tailwind.config.js` with a full design token configuration:

   **Colors (extend, do NOT override defaults):**
   - `primary`: indigo palette — `50: '#eef2ff'` through `950: '#1e1b4b'`, default `500: '#6366f1'`
   - `accent`: electric blue — `50: '#eff6ff'` through `950: '#172554'`, default `500: '#3b82f6'`
   - `voice`: emerald — `50: '#ecfdf5'` through `950: '#022c22'`, default `500: '#10b981'`
   - `social`: amber — `50: '#fffbeb'` through `950: '#451a03'`, default `500: '#f59e0b'`
   - `surface`: dark backgrounds — `900: '#0f0f23'` (existing canvas color), `800: '#1a1a2e'`, `700: '#2a2a3e'`, `600: '#3a3a4e'`
   - `glass`: `white: 'rgba(255,255,255,0.08)'`, `border: 'rgba(255,255,255,0.12)'`, `hover: 'rgba(255,255,255,0.15)'`

   **Fonts:**
   - `fontFamily.sans`: `['Inter', ...defaultTheme.fontFamily.sans]`
   - `fontFamily.heading`: `['Space Grotesk', 'Inter', ...defaultTheme.fontFamily.sans]`

   **Animations (keyframes + animation):**
   - `fadeIn`: opacity 0 to 1, 200ms ease-out
   - `fadeOut`: opacity 1 to 0, 150ms ease-in
   - `slideUp`: translateY(8px)+opacity(0) to translateY(0)+opacity(1), 200ms ease-out
   - `slideDown`: translateY(-8px)+opacity(0) to translateY(0)+opacity(1), 200ms ease-out
   - `scaleIn`: scale(0.95)+opacity(0) to scale(1)+opacity(1), 150ms ease-out
   - `ripple`: scale(0)+opacity(0.5) to scale(4)+opacity(0), 600ms ease-out
   - `float`: translateY(0) to translateY(-2px) to translateY(0), 3s ease-in-out infinite
   - `glow-pulse`: box-shadow pulse between 0px and 8px spread, 2s ease-in-out infinite

   **Box shadows (extend):**
   - `glow-sm`: `0 0 8px -2px var(--tw-shadow-color)`
   - `glow-md`: `0 0 16px -4px var(--tw-shadow-color)`
   - `glow-lg`: `0 0 24px -6px var(--tw-shadow-color)`

   **Backdrop blur (extend):**
   - `glass`: `12px` (standard glassmorphism blur)

   **Plugins:**
   - Add `require('tailwindcss-animate')` to plugins array

   Use `import defaultTheme from 'tailwindcss/defaultTheme'` at top. Keep `content` paths as-is. Export using `export default`.
  </action>
